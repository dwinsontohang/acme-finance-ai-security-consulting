# Client Requirements

Based on the challenges outlined in the problem statements, the following key requirements have been established to guide the secure and effective deployment of AI technologies across ACME Finance, ensuring the project remains focused and deliverable within a 3-month timeline.

## CTO Team's Requirements

- Establish a secure architecture for each AI product—RAG-enabled chatbots, AI Co-pilots, and Agentic AI—using consistent technical and procedural controls across all environments.
- Assess the risks and attack surface changes introduced by AI deployments to identify key exposure points and recommend effective mitigations.
- Enable safe, scalable AI adoption through clear security recommendations, staff training, and a proactive monitoring and incident response approach.

## CISO Team's Requirements

- Design a secure integration model that shows how Azure Security Copilot can work effectively with Sentinel SIEM to support core security operations.
- Provide clear, practical guidance on how to adopt and govern AI tools responsibly within the SOC, aligning with internal policies and operational needs.
- Equip the security team with the skills and tools needed to use Copilot and Sentinel confidently through focused training and actionable reference materials.
- Establish a working framework for AI-powered monitoring and response that can be quickly implemented and expanded over time.
